Overview
iperf port to Pocket PC(Windows Mobile 5.0, 6.x)


Features
 - Provides ping to check latency between nodes.
 - Provides full functionality of iperf based on 1.7.0 both client and server.
 - Logs for ping, iperf.
 - Visualization of ping latency graph. (it's good when landscape is used)
