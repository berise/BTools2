Overview
BTools is a sort of handy tools to check latency and bandwidth of your network.
Check your latency with ping and bandwidth with iperf. 
This program is based on iperf-1.7.0 and completely compatible it.


Features
 - Provides ping to check latency between nodes.
 - Provides full functionality of iperf based on 1.7.0 both client and server.
 - Logs for ping, iperf client & server.
 - Visualization of ping latency graph. (it's good when landscape is used)
